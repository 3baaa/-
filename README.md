# Member-group-file-drive-site-using-certificate-server-and-Hadoop

# 저를 포함한 팀원 4명 및 맡은 역할
저(조장)의 역할 = 하둡 서버 구축 및 웹 연동, 백엔드 개발, DB 설계
3명의 팀원의 역할 = 인증서버 구축 및 웹 연동
공통 역할 = 프런트엔드 개발, 웹 서버 구축

# 개발 관련 내용
